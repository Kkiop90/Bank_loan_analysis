Interactive Bank Loan Analysis Dashboard built in Microsoft Excel featuring data cleaning, Pivot Tables, KPI Cards, Pivot Charts, and Slicers to analyze loan performance, customer insights, and credit risk using a dataset of 5,000 records.


# 🏦 Bank Loan Analysis Dashboard (Excel)

## 📌 Project Overview

This project is an interactive **Bank Loan Analysis Dashboard** developed in **Microsoft Excel** using a dataset of **5,000 loan records**.

The objective of this project is to analyze loan performance, customer demographics, and credit risk while demonstrating practical Excel skills used in Data Analytics.

---

## 📊 Dashboard 1: Loan Performance

### KPIs
- Total Loan Applications
- Total Loan Amount
- Approval Rate
- Rejection Rate
- Average Loan Amount
- Average Interest Rate

### Visualizations
- Loan Status Distribution
- Loan Type Distribution
- State-wise Loan Amount
- Branch-wise Loan Amount
- Monthly Loan Applications

## 🧹 Data Cleaning

The dataset was cleaned before analysis by:

- Removing duplicate records
- Standardizing categorical values
- Correcting inconsistent text formatting
- Handling missing values
- Validating credit score ranges
- Removing unnecessary spaces
- Improving overall data quality

---

## 📈 Key Business Insights

- Identified states with the highest loan amounts.
- Compared loan approval and rejection trends.
- Analyzed customer demographics by age, education, and employment.
- Evaluated customer credit quality using credit scores.
- Compared branch performance based on loan distribution.
- Identified customer segments with higher default rates.
